Introducción a Sistemas de Cómputo

Introducción a los Sistemas de Cómputo es un curso de naturaleza práctica, experimental y orientada al aprendizaje a todo nivel. El curso inicia con un componente teórico conceptual de alta importancia
para el estudiante de Sistemas; ya que se establecen los conceptos informáticos más elementales, así como el componente léxico necesario para comprender los temas más profundos presentados a través
del mismo.

Proyectos de Introducción

Realizar la limpieza integral y el mantenimiento preventivo de los
componentes internos del Case, aplicando correctamente los insumos técnicos
para optimizar el funcionamiento y la refrigeración del sistema.

Este proyecto consiste en la virtualización del sistema
operativo Ubuntu 24.04.4 LTS utilizando VirtualBox
como herramienta principal. A través de este proceso se busca comprender
la importancia de la virtualización en el área informática permitiendo
ejecutar distintos sistemas operativos sin necesidad de modificar el
sistema principal del equipo. Durante el desarrollo del proyecto se
realizó la descarga de la imagen ISO de Ubuntu, la instalación del
virtualizador y la configuración de la máquina virtual asignando
recursos como memoria RAM, procesador y espacio en disco. Con esta práctica se fortalecieron conocimientos
técnicos sobre instalación, configuración y administración de sistemas operativos en entornos virtuales.

Este proyecto consistió en la fabricación de cables de red tipo patchcord, elaborando un cable directo y un cable cruzado mediante el uso de herramientas de ponchado y conectores RJ45. Durante la práctica se aplicaron las normas de cableado T568A y T568B, organizando correctamente los colores de los hilos para garantizar una conexión funcional. Además, se realizó el proceso completo de corte, pelado, ordenamiento y ponchado del cable UTP, finalizando con pruebas en el tester para verificar el correcto funcionamiento de cada cable. Este laboratorio permitió fortalecer conocimientos prácticos sobre redes y hardware, así como desarrollar habilidades técnicas en la creación y verificación de conexiones de red.

Este proyecto consistió en la configuración de una red local (LAN) utilizando un switch y cables de red directos para interconectar tres computadoras. Durante la práctica se asignaron direcciones IP estáticas a cada equipo, permitiendo establecer comunicación entre las computadoras mediante pruebas de ping y verificando la conectividad de la red. Además, se configuró un mismo grupo de trabajo y se habilitó el uso compartido de archivos y carpetas, facilitando el intercambio de información entre los equipos conectados. Este proyecto permitió fortalecer conocimientos prácticos sobre redes, direccionamiento IP, configuración de recursos compartidos y comunicación entre dispositivos dentro de una red local.

Este proyecto consiste en diseñar e implementar una red LAN en una casa de tres niveles, conectando distintos dispositivos mediante red alámbrica e inalámbrica. Se configurará un router con seguridad y direcciones IP, un switch en cascada y un repetidor WiFi para permitir la comunicación entre computadoras, laptops, tablets e impresoras. Además, se realizarán pruebas de conectividad entre dispositivos y se documentará todo el proceso en un informe PDF y un video explicativo del funcionamiento de la red.

Este proyecto consiste en el análisis y diseño de bases de datos para distintos escenarios utilizando diagramas entidad-relación (ER). Se desarrollarán modelos para una biblioteca, una tienda y una red social, identificando las entidades principales y sus relaciones para posteriormente crear las bases de datos y tablas en PHPMyAdmin. Además, se generarán los scripts SQL correspondientes, se insertarán registros de prueba y se visualizarán los diagramas ER dentro del diseñador de PHPMyAdmin. Todo el proceso deberá documentarse en un informe PDF con capturas de pantalla y enlaces a los archivos generados en Drive.
